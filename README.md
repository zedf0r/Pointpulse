# О проекте

Список задач с бесконечным скроллом, виртуализацией и полным **CRUD**-взаимодействием. Оптимизировано для производительности при большом количестве задач.

## Быстрый старт

1. Установить зависимости `npm i`
2. Запустить сборку и сервер `npm run start`

После успешной сборки веб-сайт можно будет посмотреть и протестировать по адресу `http://localhost:3000`

## Стек Технологий

- **Frontend**: React 19, TypeScript
- **Роутинг**: React Router DOM v6
- **Стили**: SCSS Modules
- **API**: Tanstack Query, json-server (Mockoon)
- **Бандлер**: Vite
